## 1 Задание к работе
1.1. Разработать программу на языке высокого уровня Python для составления продукционной системы и прямого вывода на основе данной системы:
1) Программа предоставляет интерфейс для добавления требуемого числа правил, включая условия и заключения. Предусмотрено 3 вида условий: простое условие, составное с И, составное с ИЛИ.
2) Полученная продукционная система выводится на экран.
3) На основе введенных исходных данных реализуется процесс прямого вывода и выводятся результаты по каждому правилу.

1.2. Продемонстрировать процесс тестирования:
1) Добавление не менее чем 5 правил, связанных с темой планируемой ВКР в магистратуре (Тема: **Выбор архитектуры базы данных**).
2) Привести два примера прямого вывода. В каждом примере должны стать активными не менее 2-х правил.
