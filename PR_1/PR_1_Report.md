## 1 Задание к работе
I. Разработать программу на языке высокого уровня Python для решения практической задачи, связанной с консалтингом в области выбора профессии для последующего обучения и получения соответствующей специальности.

Введем следующие обозначения:
* $X$ – множество специальностей, по которым производится набор на обучение;
* $Y$ – множество психофизиологических характеристик;
* $Z$ – множество кандидатов на обучение.

II. В программе должны выполняться следующие действия:
1) Определение значений функции принадлежности нечеткого отношения $S$, построенного на множествах $X$ и $Y$, описывающего психо-физиологическое профилирование специальностей (случайными числами). Вывод на экран в форме матрицы.
2) Определение значений функции принадлежности нечеткого отношения $T$, построенного на множествах $Y$ и $Z$, описывающего профилирование кандидатов (случайными числами). Вывод на экран в форме матрицы.
3) Определение нечеткой композиции отношений $S$ и $T$ с целью получения рекомендаций.
4) Определение рекомендуемых специальностей для имеющихся кандидатов по максимумам.
5) Определение наиболее подходящих кандидатов для обучения по каждой специальности по максимумам.
